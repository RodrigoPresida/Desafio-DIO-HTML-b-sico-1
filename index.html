Principais Tags

Tags de estrutura

  • html: Define o início e o fim do documento HTML.
  • head: Define informações sobre o documento.
  • body: Define o corpo do documento.
  • title: Define o título da página, exibido na barra de título do navegador.
  • meta: Fornece metadados sobre o documento, como descrição.

Essas tags compoem a estrutura básica de um documento HTML.

Tags de Titulo e sub-titulo

  • h1: Define o título do documento.
  • h2: Define um subtítulo do documento.
  • h3: Define um sub-subtítulo do documento.
  • h4: Define um sub-sub-subtítulo do documento.
  • h5: Define um sub-sub-sub-subtítulo do documento.
  • h6: Define um sub-sub-sub-sub-subtítulo do documento.

Essas tags são uteis para criar titulos e sub-titulos.

Tags para formatar textos

  • p: Define um parágrafo.
  • br: Insere uma quebra de linha.
  • hr: Insere uma linha horizontal.
  • strong: Define um texto em negrito.
  • em: Define um texto em itálico.
  • u: Define um texto sublinhado.
  • blockquote: Define um texto em forma de citação.
  • sub: Define um texto subscrito.
  • sup: Define um texto sobrescrito.

Essas tags são uteis para formatar textos
